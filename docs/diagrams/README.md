# Workflow Diagrams

This folder contains Mermaid workflow diagrams for the HackNation AI Scientist demo.

- `overall-workflow.md`: end-to-end product and data flow
- `rachael-agent.md`: scientific QC, keyword search, embeddings, and literature ranking
- `eric-agent.md`: protocol extraction, inventory derivation, supplier search, and budgeting
- `faith-agent.md`: final plan synthesis and feedback caching

These diagrams are written in Markdown so they render directly on GitHub.
